🧗‍♂️ Le mie scalate

Questo è il mio sito personale, un diario di arrampicata dove raccolgo racconti, pensieri ed esperienze vissute in montagna.
Il sito è scritto interamente in HTML + CSS e pubblicato tramite GitHub Pages.

🚀 Demo

🔗 Visita il sito online

https://andreabdeluca.github.io/websitedemo/

📖 Contenuti

Racconti delle mie uscite in falesia e in montagna.

Diario con date, luoghi e riflessioni personali sulle pareti ma anche su quello che ci sta intorno.

🛠️ Tecnologie usate

HTML5

CSS3

Nessun framework: puro codice statico (tutto scritto da un informatico alle prime armi).

📂 Struttura

index.html → pagina principale con racconti e diario.

logo-*.png → icone usate nel footer.

🔧 Installazione e utilizzo

Se vuoi aprire il sito in locale:
git clone https://github.com/tuonome/tuorepo.git
cd tuorepo
open index.html
